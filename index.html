Step-by-Step Guide

1. Default Classes

Your school has been automatically set up with default classes:

  • Primary One, Primary Two, Primary Three, Primary Four, Primary Five, Primary Six, Primary Seven
  • Senior One, Senior Two, Senior Three, Senior Four, Senior Five, Senior Six

2. Add Students

Go to the Students tab to upload your student lists using Excel files.

3. Manage Subjects

Visit the Subjects tab to view and manage your school's subjects.

4. Enter Marks

Use the Enter Marks tab to input student performance data.

5. Generate Reports

Finally, go to the Reports tab to view and export student performance reports.
